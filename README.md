<h3></h3>

Sample Python application on Django with PostgreSQL database.

<h3>Requirements</h3>

____


- django 4.0.1
- Pillow 9.0.0
- psycopg2-binary 2.9.3
- django-prometheus 2.2.0
- uwsgi 2.0.20

<h3>Deployment</h3>

____

* Build and run docker-compose:
```shell
docker-compose up
```

* Enjoy!
